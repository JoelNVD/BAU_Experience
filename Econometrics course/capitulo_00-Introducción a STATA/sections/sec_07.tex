\begin{frame}[fragile]{\texttt{Help} y \texttt{Search}}
	Stata tiene un útil comando de búsqueda; Es un buen punto de partida para buscar definiciones y conceptos relacionados.
\begin{lstlisting}[language=Stata, numbers=none]
search regression
searcs ols
\end{lstlisting}
	Stata tiene un comando de ayuda útil si ya conoce el nombre del comando que necesita
\begin{lstlisting}[language=Stata, numbers=none]
help regress
\end{lstlisting}
	El comando \colorbox{codegray}{\texttt{help}} muestra la sintaxis del comando. Algunos comandos básicos que debe conocer son: \colorbox{codegray}{\textcolor{codeblue}{ \texttt{regress}}} y \colorbox{codegray}{\textcolor{codeblue}{\texttt{test}}}, que realizan la estimación de MCO y las pruebas de significación respectivamente.
\end{frame}
